How it works
01
reset()
Call reset() with a task ID. Get back an Observation — buggy code, error message, stack trace, context. Episode begins.
02
step(action)
Agent takes an Action — diagnose, suggest fix, ask for info, or submit solution. Environment returns reward + feedback immediately.
03
reward signal
Partial rewards at every step. Right diagnosis = +score. Correct fix = +score. All tests pass = full score. Wrong direction = 0.
04
grade()
Final score via /grader. Deterministic — same actions always produce same score. Run /baseline to benchmark any LLM agent.
